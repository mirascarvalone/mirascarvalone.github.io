Week 4: Ruby - Arrays and Hashes

November 13, 2015

This week we learned about arrays and hashes in Ruby, and got a sense of some examples where we would use both of these concepts in our programs.

Arrays

Arrays are collections of elements enclosed by brackets and separated by commas. The elements can be numbers, strings, integers, etc. Elements of an array are indexed at zero, and if we want to access a particular element of an array, we can write array[position]. In the example below, we could enter the following to obtain the first element of the array (1): 

      my_numbers = [1,2,3,4,5]
      puts my_numbers[0]

Hashes

Hashes are a collection of pairs, called keys and values. Each key maps to one value, and the pairs in a hash are not indexed, which is different than an array. For example, the array below maps each month of the year to its numerical value. If we want to know what month March is, we type the following: 

      months = {
        'January'     => 1,
        'February'  => 2,
        'March'      => 3,
        'April'      => 4,
        'May'   => 5,
        'June'    => 6,
        'July'     => 7,
        'August'  => 8,
        'September'      => 9,
        'October'      => 10,
        'November'   => 11,
        'December'    => 12
      }

      months['March']
