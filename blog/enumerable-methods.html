The Map Enumerable Method

The map method calls a block of code on every element of the original object and outputs that to a new array. For example, we could use the map method to add 1 to each number in an array: 

      numbers = [1,2,3,4,5]
      numbers.map {|x| x + 1}

      => [2, 3, 4, 5, 6]

Map can also work on a range, for example: 

      (1..3).map {|i| i + 5}

      => [6, 7, 8]

Map can also work on an array of strings, for example: 

      names = ["Rand","Donald","Hillary"]
      names.map {|x| x.upcase}
      => ["RAND", "DONALD", "HILLARY"]

Part of the usefulness of the map method is that it automatically returns an array, which adds functionality over the each method.
